## 🚀 ¿Cómo usar estos reportes?
 
1. **Descarga** el archivo `.pbix` que desees explorar
2. **Instala Power BI Desktop** (gratuito) desde el sitio oficial:
   👉 [https://powerbi.microsoft.com/desktop](https://powerbi.microsoft.com/desktop)
3. **Abre** el archivo descargado desde Power BI Desktop
4. ¡Listo! Podrás explorar todas las visualizaciones e interactuar con los filtros
